# Printpal
>> Project is still in development. More features to be added.

Printpal is a useful app for those passionate in 3d printing. The app helps you track how many filament spools you have left (including their colors and types), store models you enjoy and links to them, save crucial information about your printers and their maintenance.

## Features

* Save data about your 3D Printers
* Track how many spools of filament you have left
* Keep up with the print jobs
